# 🧩 Японский Кроссворд (Nonogram Editor & Solver)

## 📋 Описание
Полнофункциональное приложение для создания, редактирования и решения японских кроссвордов (nonograms) с интуитивным графическим интерфейсом и интеллектуальным алгоритмом решения.

---

## ✨ Основные возможности

### 🎨 **Визуальный редактор**
- **Динамическое создание полей** любого размера (до 150×150 ячеек)
- **Настраиваемый размер ячеек** от 10 до 20 пикселей
- **Три визуальные области**:
  - Верхняя панель: задания для столбцов
  - Левая панель: задания для строк
  - Основное поле: область для решения
- **Автоматическое масштабирование** интерфейса под размер кроссворда

### 🛠️ **Инструменты редактирования**
- **Интеллектуальный ввод** заданий с контекстными подсказками
- **Быстрое копирование** значений между строками и столбцами
- **Массовое заполнение** крестиками или закрашиванием
- **Цветовая индикация** проверенных заданий
- **Циклическое переключение** состояний ячеек

### 💾 **Работа с файлами**
- **Собственный формат `.jcw`** с использованием INI-файлов
- **Автосохранение** настроек и прогресса
- **Импорт/экспорт** кроссвордов
- **Контекстные названия** файлов в заголовке окна

### 🧠 **Алгоритм решения**
- **Интеллектуальная проверка** корректности заданий
- **Автоматическое определение** обязательных ячеек
- **Двухпроходный алгоритм** анализа строк и столбцов
- **Определение несовместимых** комбинаций
- **Пошаговое решение** с визуализацией процесса

### 🔍 **Проверка и валидация**
- **Контроль суммы** значений по строкам и столбцам
- **Проверка вместимости** заданий в поле
- **Обнаружение противоречий** в введенных данных
- **Визуальные предупреждения** о проблемах

## 🚀 Быстрый старт

### 📋 **Системные требования**
- **ОС**: Windows 7/8/10/11
- **Delphi**: 10.4+ (для компиляции)
- **Память**: 512 MB RAM
- **Дисковое пространство**: 10 MB

### 🔧 **Установка**
1. Клонируйте репозиторий или скачайте архив
2. Откройте проект `JapaneseCrossword.dpr` в Delphi
3. Скомпилируйте проект
4. Запустите полученный исполняемый файл

## 📖 Руководство пользователя

### 📝 **Создание нового кроссворда**
1. В меню выберите "Кроссворд" → "Создать"
2. Укажите размеры верхней и левой панелей
3. Выберите размер ячеек
4. Нажмите "Создать"

### ✏️ **Редактирование заданий**
- **Левый клик** на ячейке задания: отметить как проверенное
- **Правый клик** на ячейке задания: ввести числовое значение
- **Левый клик** на поле решения: циклически изменить состояние ячейки
- Используйте иконки копирования для быстрого заполнения

### 💾 **Сохранение и загрузка**
- **Сохранить**: Файл → Сохранить (Ctrl+S)
- **Сохранить как**: Файл → Сохранить как... (Ctrl+Shift+S)
- **Открыть**: Файл → Открыть (Ctrl+O)

### 🧩 **Решение кроссворда**
- **Проверить задание**: Кроссворд → Проверить задание
- **Авторешение**: Кроссворд → Разгадать
- **Очистить поле**: Кроссворд → Очистить

## 🏗️ Архитектура проекта

### 📁 **Структура проекта**
``` text
JapaneseCrossword/
├── Основная форма (TBasic_Frm) # Главный интерфейс
├── Форма создания (TCreate_Frm) # Настройка размеров
├── Модуль функций (Functions_Unit) # Типы данных и алгоритмы
└── Исполняемый файл # Скомпилированное приложение
```

### 🧬 **Ключевые структуры данных**
- **TTaskCell**: Ячейка задания с координатами, значением и состоянием
- **TResultCell**: Ячейка результата с типом заполнения
- **TTaskField/TResultField**: Двумерные массивы для хранения поля
